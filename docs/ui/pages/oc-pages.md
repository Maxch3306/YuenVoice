### Components

| Element | shadcn Component | Notes |
|---------|-----------------|-------|
| Year tabs | `Tabs` | Filter by year |
| Type filter | `Select` | 5 options |
| Document card | `Card` | File icon + type badge + title + date |
| Type badge | `Badge variant="outline"` | Color per type (meeting=blue, finance=green, resolution=amber, notice=slate) |
| Upload button | `Button variant="default"` | Conditional on role |
| Upload dialog | `Dialog` | Form with file picker |
| File picker | `Input type="file"` | Styled as dashed zone |
| Pagination | Custom | Simple prev/next |

### Components

| Element | shadcn Component | Notes |
|---------|-----------------|-------|
| Metadata card | `Card` | Type badge, title, publisher, date, description |
| Download button | `Button variant="outline"` | With download icon |
| Delete button | `Button variant="destructive"` | OC/Admin only, opens `AlertDialog` |
| PDF viewer | `iframe` or `react-pdf` | `w-full h-[70vh] rounded-md border` |
| Page nav | Custom | Page number + prev/next buttons |
